# JuriSight Project Requirements
# Note: This is a Node.js/TypeScript project, not a Python project.
# To install project dependencies, use `npm install` instead of `pip install -r requirements.txt`.

# --- Core System Requirements ---
Node.js >= 18.0.0
npm >= 9.0.0

# --- Databases & Services ---
PostgreSQL (Neon cloud database or local instance)
Redis (For caching and queues - optional but recommended)

# --- Third-Party API Keys ---
# At least one AI provider is required for the application to function:
OpenAI API Key (Primary for document analysis & chat)
Google Gemini API Key (Fallback for AI services)
Groq API Key (Additional fallback for AI services)

# --- Infrastructure (Optional for Development) ---
Docker
Docker Compose

# --- Setup Instructions ---
# 1. Run `npm install` in the root directory to install all workspaces.
# 2. Copy `.env.example` to `.env` and configure your database and API keys.
# 3. Run `npx prisma generate` and `npx prisma db push` in apps/backend.
# 4. Run `npm run dev` to start the development servers.
